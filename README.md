# CellereumRR

This is a remake of my old Telegram bot, Cellereum.

The original project was abandoned in early 2021, but at the time it served as a simple currency system and central hub for several bots I planned to develop. The long-term goal was to build a bot network called xBot, with Cellereum acting as its main hub.

This was one of the first programs I ever wrote in C#, and it is very poorly written in many places, especially considering I was only 13 years old when I made it.

I decided to publish it on GitHub because it has significant sentimental value to me. It represents the beginning of my journey into programming and serves as a snapshot of how I learned and experimented at the time.

Also, keep in mind that this is the remake version. The original project was *far* worse than this one. Seriously.
